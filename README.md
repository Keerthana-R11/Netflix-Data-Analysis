## Netflix-Data-Analysis
Exploratory data analysis on Netflix movies and TV shows dataset using Python.

## 📌 Project Overview
This project analyzes Netflix movies and TV shows dataset to find insights
such as content type, country distribution, ratings, and trends.

##  🔗Tools Used
- Python
- Pandas
- Jupyter Notebook
- SQL
- Power BI
  
## 📂 Dataset
Netflix Movies and TV Shows dataset from Kaggle.
